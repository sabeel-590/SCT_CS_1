## Project Overview

The Caesar Cipher Encryption & Decryption Tool is a Python-based GUI application developed as part of the SkillCraft Technology Cyber Security Internship.

This project demonstrates the implementation of the classical Caesar Cipher cryptographic technique, which encrypts and decrypts text by shifting alphabetic characters by a user-defined value.

The application provides a simple and user-friendly graphical interface that allows users to securely encrypt and decrypt messages using the Caesar Cipher algorithm.



## Features

- Encrypt plain text messages
- Decrypt encrypted messages
- User-defined shift value
- Interactive Graphical User Interface (GUI)
- Dark-themed professional design
- Handles uppercase and lowercase letters
- Preserves spaces, numbers, and special characters
- Easy-to-use interface
- Real-time encryption and decryption results
